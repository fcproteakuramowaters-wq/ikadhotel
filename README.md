# IKAD Hotel Booking System

A Django-based hotel booking system for IKAD Hotels, featuring two locations:
- IKAD Hotel & Suite - Victoria Island
- IKAD Hotel & Suite - Cooli

## Features

- Online room booking system
- Multiple hotel location support
- Room type selection
- Date-based availability
- Email confirmation system
- Responsive design
- Special request handling

## Tech Stack

- Python 3.x
- Django
- Bootstrap 5
- JavaScript
- SQLite (development)

## Setup

1. Clone the repository
```bash
git clone [repository-url]
cd ikadhotel
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Apply migrations
```bash
python manage.py migrate
```

4. Run the development server
```bash
python manage.py runserver
```
